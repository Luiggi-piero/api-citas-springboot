## Descripción del proyecto
API dedicada a la administración de citas y usuarios

## Funcionalidades
- [x] Crear administrador
- [x] Crear agente
- [x] Crear cliente
- [x] Crear cita
- [x] Asignar cita
- [x] Completar cita
- [x] Obtener citas pendientes del agente
- [x] Obtener cita completada por id
